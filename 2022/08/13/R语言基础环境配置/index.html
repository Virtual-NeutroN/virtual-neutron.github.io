R语言基础环境配置

发表于2022-08-13|更新于2025-03-21|网站相关
|字数总计:970|阅读时长:3分钟|阅读量:
  • Hey阿达西！R语言这货啊，就是个统计界的老登，专治各种数据不服！你想象一下：当你面对Excel表格像看天书的时候，R抄起键盘就能把数据拧成麻花——算平均数？画骚气折线图？预测明天奶茶店会不会爆单？它连眼皮都不带眨的！
  • 这家伙长得挺猎奇，动不动就爱用”<-“这种火星符号赋值（对，就是那个长得像鼻孔朝左的箭头）。刚开始你可能想砸键盘：”为啥不用等号？！”，但用久了就会发现…其实还是想砸键盘（，但至少能跟它混熟了！最骚的是它的画图能力！ggplot2包一出手（本人不知道是什么），什么柱状图热力图，分分钟给你整得比网红自拍还精致。老板看了直呼”拿来吧你”，转头就塞进PPT里装B。

安装

VSCODE 安装

  1. 进入Visual Studio Code官网点我进入
  2. 直接点击网页上大大的download
  3. 打开安装程序，无脑一直next

    如果您对文件安放位置有要求，请根据自己的需求安装位置

  4. 安装完毕打开软件，开始配置中文

VSCODE 配置

  1. 进入页面之后，按下ctrl+shift+p
  2. 在弹出的搜索框中输入configure language，然后选择搜索出来的Configure Display Language
  3. 此时看向左边弹出的栏，找到简体中文，然后install，等待安装完毕
  4. 重启软件（大退，结束进程那种），即可

R语言环境安装

  1. 进入R-project官网下载点我直接进入下载界面（这里其实用的是清华镜像网站）
  2. 直接点击网页上大大的download
  3. 打开安装程序，无脑一直next

    如果您对文件安放位置有要求，请根据自己的需求安装位置

  4. 安装完毕，环境配置成功

VSCODE 部署 R

  1. 打开VSCODE，进入页面之后，按下ctrl+shift+x
  2. 在搜索栏中搜索R，选择第一个（见下图），点击install
  3. 安装好后在搜索栏中搜索R Debugger，选择第一个（见下图），点击install
  4. 全部安装完毕后重启VSCODE

创建工程文件夹

  1. 电脑上找一个安放你R语言文件的地方，新建文件夹
  2. 打开VSCODE，选择上方菜单栏 文件->打开文件夹->选择你刚刚新建的文件夹
  3. 选择上方菜单栏 文件->新建文本文件->点击选择语言->选择R
  4. 保存文件（可以ctrl+s），把这个文件放入你刚刚新建的文件夹里面

Rdebugger 安装vscdebugger

  1. 回到VSCODE页面，按下ctrl+shift+p
  2. 在弹出的搜索框中输入r.debugger.updateRPackage，选择第一个选项
  3. 等待安装，大概率网会不好，多下几次，失败了就重试，直到这几个都能成功安装

  4. 至此，全部部署任务结束
